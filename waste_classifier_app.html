Custom 4-block CNN trained on the Recyclable & Household Waste Classification dataset with 30 categories. Each block uses Conv2D → BatchNorm → ReLU → MaxPool. The classifier head uses GlobalAveragePooling + Dense + Dropout + Softmax. Best weights saved via ModelCheckpoint on validation accuracy.
